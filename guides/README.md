# Guides

Learning paths, decision trees, and reference material for testers at all levels.

## For New Testers

- **learning-path.md** — Where to start, how to navigate this repo, key milestones
- **glossary.md** — Terminology: oracle, executor, MCP, agentic vs. scaffolded, etc.

## For Decision Making

- **decision-tree-what-to-automate.md** — Given a feature, should you automate it? With AI or traditional tools? Unit test, API test, or E2E? How much should the oracle be detailed?

## How to Use These Guides

Start here if:
- You're new to this repo
- You're unsure which skill to pick
- You want to understand the concepts before diving into prompts
- You're deciding whether to automate a specific test

Then move to `/concepts` for depth, `/prompts` for specifics, and `/skills` to run something.
